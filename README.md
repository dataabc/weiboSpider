# 来源

本项目受`weiboSpider`[](https://github.com/dataabc/weiboSpider)启发并fork，增加一些功能

# 增加功能
- python3 支持 
- 增加 发布设备
- 增加对于原创和转发的识别
- 对于微博全文可以获取
- 对于微博内容中链接增加获取，并用$$进行包围
- 识别置顶微博，和0微博
- 写入csv文件

# 注意事项
- 对全文的获取是通过识别链接跳转到新页面获取，因此需要注意延迟，避免被封
- 转发理由的获取是通过`beautifulsoup4`模块获得
- csv文件写入是通过`pandas`模块获得
